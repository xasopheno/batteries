# My archlinux setup scripts

### Download:

```shell
wget https://raw.githubusercontent.com/xasopheno/batteries/master/archlinux/setup/init
wget https://raw.githubusercontent.com/xasopheno/batteries/master/archlinux/setup/chroot
```

### Verify:

```shell
sha1sum init chroot
```

### Install:

```shell
bash init
```
